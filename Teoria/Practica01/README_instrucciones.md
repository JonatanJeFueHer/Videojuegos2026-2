Archivo de instrucciones de funcionamiento del juego tipo pong con OpenGL.
- Como compilar el proyecto:
	- Seguir la guĒa https://www.geeksforgeeks.org/installation-guide/how-to-setup-opengl-with-visual-studio-2019-on-windows-10/ para configurar el entorno de desarrollo con OpenGL.
	- Descargar el c¾digo fuente del proyecto.
- Como ejecutar el juego:
	- Abrir el proyecto en Visual Studio.
	- Compilar el proyecto (Ctrl + Shift + B).
	- Ejecutar el proyecto (F5).
	
	Alternatrivamente, se puede ejecutar el archivo Project1.exe que viene en el repositorio descargado en la ruta "...\Videojuegos2026-2\Teoria\Practica01\Debug"
	
	- Usar el comando "cd" para navegar a la carpeta donde se encuentra el archivo ejecutable.
	- Teclear el nombre del archivo ejecutable (Project1.exe) y presionar Enter para ejecutar el juego.
- Controles del juego:
	- Usar las teclas "W" y "S" para mover la paleta izquierda (Jugador1) hacia arriba y hacia abajo, respectivamente.
	- Usar las teclas "Up Arrow" y "Down Arrow" para mover la paleta derecha (Jugador 2) hacia arriba y hacia abajo, respectivamente.
	- El objetivo del juego es evitar que la pelota pase por tu paleta y tratar de hacer que pase por la paleta del oponente para ganar puntos.
- Dependencias necesarias:
	- OpenGL
	- GLU
	- GLUT
