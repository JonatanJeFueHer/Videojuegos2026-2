# README.md - Instrucciones para Ejecutar el Proyecto Practica02_3D en Unity

## Requisitos Previos
Instala Unity Hub desde el sitio oficial de Unity, que gestiona las versiones del 
editor. Luego, descarga e instala la versión 2022.3.7f1 de Unity Editor, que es 
compatible con este proyecto. Asegúrate de incluir los módulos necesarios para tu 
plataforma (Windows, macOS, etc.) durante la instalación.

## Abrir el Proyecto
-Lanza Unity Hub y ve a la pestaña "Proyectos" 
-Agrega el proyecto desde el disco con el botón Add y selecciona "add project from 
disk"
-Cuando termine de cargar el proyecto haz clic en "Practica06_3D"
-Espera a que se cargue la escena

## Ejecutar en el Editor
Una vez abierto, presiona el botón **Play** (triángulo ▶️) en la barra superior para 
ejecutar la escena actual en el modo de juego dentro del Editor. 
Usa **Pause** (⏸️) para pausar y **Stop** (⏹️) para detener. Verifica la consola 
para errores o mensajes de debug en scripts o assets.

Al hacer click en Play se cargará la escena y tendras control sobre el personaje 
jugabe, muevete con W,A,S,D, salta con barra espaciadora y mueve el ratón hacia donde 
quieras que el personaje se mueva.
