# README.md - Instrucciones para Ejecutar el Proyecto Practica02_3D en Unity

## Requisitos Previos
Instala Unity Hub desde el sitio oficial de Unity, que gestiona las versiones del editor.Luego, descarga e instala la versión 6000.3.7f1

## Abrir el Proyecto
-Lanza Unity Hub y ve a la pestaña "Proyectos" 
-Agrega el proyecto desde el disco con el botón Add y selecciona "add project from disk"
-Cuando termine de cargar el proyecto haz clic en "Practica02_3D"
-Espera a que se cargue la escena

## Ejecutar en el Editor
Una vez abierto, presiona el botón **Play** (triángulo ▶️) en la barra superior para ejecutar la escena actual en el modo de juego dentro del Editor. Usa **Pause** (⏸️) para pausar y **Stop** (⏹️) para detener. Verifica la consola para errores o mensajes de debug en scripts o assets.

Al hacer click en Play se cargará la escena y el agente se empezara a mover aleatoriamente por el mapa, y si se hace click en algun punto del mapa el agente ira hacia esa ubicación y después seguirá una ruta aleatoria. Si se hacen clicks en distintos puntos del mapa de manera seguida, el agente iniciara la ruta para después ignorar el primer destino y dirigirse al segundo destino y así sucecsivamente.
